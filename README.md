SMS
===

(SMS)Subscriber Management System Module developed in Java/ZK framework

This is a Java/ZK based web-application in Linux environment, the main application comprises of many modules such as LCO
(Local Cable Operator) Management, Multi CAS Mediation & Management,Multi Tax System, Billing, Inventory Management and 
Complain Management.
